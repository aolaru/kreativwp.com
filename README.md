# kreativwp.com

Official static website for [kreativwp.com](https://kreativwp.com).

## Overview

This repository contains the landing page for Kreativ WP, including:

- plugin showcase content
- SEO metadata and structured data
- favicon and branding assets
- responsive styles

## Files

- `index.html` - page content and metadata
- `style.css` - site styling
- `favicon.svg` - site favicon
- `CNAME` - custom domain for GitHub Pages

## Deployment

Designed for GitHub Pages with the custom domain `kreativwp.com`.

## Visual Regression Checks

- Install deps: `npm install`
- Run screenshots: `npm run test:visual`
- Outputs are saved to `tests/home-dark.png` and `tests/home-light.png`.
